Music game based on [Open Hexagon](https://github.com/SuperV1234/SSVOpenHexagon) and [Super Hexagon](http://superhexagon.com/)

To run the game, type:
```
gradlew maps:bundle (on Windows)
./gradlew maps:bundle (on Unix/Linux)
```
Then:
```
gradlew desktop:run (on Windows)
./gradlew desktop:run (on Unix/Linux)
```
